## Calculator App

A calculator app is a software application that performs arithmetic operations on numbers. It provides a user interface for entering values and operators, and displays the computed result.

### Core Features

- **Basic arithmetic**: addition, subtraction, multiplication, and division
- **Input handling**: accepts numeric input from the user
- **Output display**: shows the result of calculations
- **Clear/reset**: allows users to start a new calculation

### Todos

- Decimal and negative number support
- Order of operations (parentheses, precedence)
- Advanced functions (square root, exponents, percentages)
- Calculation history
